# Timestamp

## Description
Backend APIs convert time to unix timestamp.

## Use
```
git clone https://github.com/hadinhtu97/timestamp
cd timestamp
npm install
npm run start
```

## API
* GET
  * `[]/api/timestamp/[date-time]`
  * `[]/api/timestamp/[unix-time]`
